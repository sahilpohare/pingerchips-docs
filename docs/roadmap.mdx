# Pingerchips Roadmap

## 2023

-   Stabilse the current release and launch v1.0
-   Introduce Pingercritter IOT (MQTT) networking

## 2024 Q1

-   Introduce Branch node to allow branching of the workflow.
-   Introduce Join node to allow joining of the workflow.
-   Introduce Kafka as a node to allow publishing to Kafka.
-   Introduce a new node to allow publishing Apache Spark jobs.
